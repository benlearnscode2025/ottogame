# Home Stretch: Night Shift

A fast, tactile showroom survival game built with HTML, CSS, and the Canvas API.

![Home Stretch title screen](assets/body-front.png)

## Play locally

Open `index.html` in a modern browser or double-click `play-home-stretch.bat` on Windows.

## Controls

- **WASD / arrow keys:** move
- **Mouse:** aim
- **Space:** dash
- **P / Escape:** pause

## Deploy to Vercel

1. Import this GitHub repository in Vercel.
2. Select **Other** as the framework preset if Vercel asks.
3. Leave the build command empty and deploy from the repository root.

The game is a static site and requires no environment variables or server runtime.
